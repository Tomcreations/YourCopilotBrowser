Welcome to YCB Browser

A fast, private, WebView2-powered browser built for Windows. Here's everything you need to know to get started.

Version 1.0

Core Features

Tabs

Open multiple tabs with Ctrl+T. Middle-click a link to open it in a new tab. Click the × to close. Switch tabs by clicking them.

Address Bar

Type a URL or search query and press Enter. It automatically detects URLs vs. searches using your chosen search engine.

Bookmarks

Click the ★ star icon in the address bar to bookmark any page. Click it again to remove the bookmark. Saved bookmarks appear in the bookmarks bar (enable in Settings).

Password Manager

YCB will ask to save your password when you log in to a site. On return visits, click the password field and your saved credentials appear. Manage them via ⋮ → Password Manager.

Downloads

Files download silently in the background — no Edge pop-up. A shelf appears at the bottom showing progress. View all downloads via ⋮ → Downloads or Ctrl+J.

Audio Indicator

When a tab is playing audio, a 🔊 speaker icon appears on that tab so you can instantly see which tab has sound.

Incognito Mode

Open a private window with Ctrl+Shift+N. History, cookies and passwords are never saved. An incognito bar is shown at the top as a reminder.

Settings

Access Settings via ⋮ → Settings. Change search engine, theme (dark/light), home page, startup behaviour, and bookmarks bar visibility.

Keyboard Shortcuts

Ctrl+T

New Tab

Open a fresh tab with your home page or new tab page.

Ctrl+W

Close Tab

Close the currently active tab. Closes the window if it's the last tab.

Ctrl+H

History

Browse your full browsing history and search through past visits.

Ctrl+J

Downloads

Open the downloads page to see all your downloaded files.

Ctrl+L

Focus Address Bar

Jump to the address bar to type a new URL or search query.

Ctrl+Shift+N

Incognito Window

Open a new private window — nothing is saved to history or passwords.

Alt+←

Back / Forward

Navigate back with Alt+← and forward with Alt+→.

Ctrl+R

Reload

Refresh the current page. Use Ctrl+Shift+R to force-reload (bypass cache).

Ctrl++ / Ctrl+-

Zoom In / Out

Increase or decrease the zoom level of the current page.

Password Manager Tips

Saving passwords

When you submit a login form, a bar appears at the top asking "Save password for site.com?". Click Save to store it securely.

Autofill

On any login page for a saved site, click the password field. A dropdown shows your saved accounts — click one to fill in your username and password automatically.

Domain matching

Passwords match by domain, not exact URL. A password saved on github.com/login will autofill on any github.com page.

Master password

The Password Manager is protected by a master password. Set one on your first visit. You'll need it each time you open the manager.
