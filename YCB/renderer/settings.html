Diagnostic Logging
Send diagnostic data
Helps us detect crashes and improve stability
What we log
All data is anonymous, non-personal and used only for debugging:
  • App launch & WebView2 version — to detect startup failures
  • Navigation errors (domain only, never full URLs) — to find broken sites
  • Tab open/close counts — to detect memory leaks
  • Download outcomes (file type & size, no filenames) — to catch download bugs
  • Password prompt shown/saved (domain only, never credentials) — to debug autofill
  • UI freeze & crash reports — to fix hangs
💙
We'd appreciate you keeping this on.
Diagnostic data is the only way we can find and fix issues — we do not log your browsing history, passwords, URLs you visit, personal files, or any identifying information. This data is never used for advertising, tracking, or any malicious purpose.
